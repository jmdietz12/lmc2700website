Coding Projects Website

A fun look into some past coding projects, including interactive elements!

Use the up and down arrow keys to move through the website!

Website 1: Breath of the Wild Compendium

Using public API and Streamlit in CS 1301, I was able to make a useful website that displays weapons, items, etc. and tell stats about them in The Legend of Zelda: Breath of the Wild. I included 3 search features: by name, by item number, or randomly by category.

Website 2: Harry Potter/Georgia Tech Sorting Quiz

I created a "Buzz Feed" style form in CS 1301 that would ask questions about Georgia Tech related things and sort the user into a Hogwarts house. I used different html form features and if statements to determine the house.

Game: Galactic Robotz

Using an online course in high school, I was able to design and partially program a platformer game in Unity. It utilizes enemies, collectables, and other platforming elements. It has some bug problems due to the hardware limitations I had at the time.

Program: Atlanta Braves Lightsaber

The coding was light for this, only having to add in a prebuilt line of code to determine the color and timing. I converted videos and sound bites into .wav files to the specific kbps needed by the lightsaber using DiVinci Resolve.
